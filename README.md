# IT-S1. Introduction

A system security audit was conducted to evaluate the safety and protection of the computer system. The purpose of this audit was to check important security areas such as Encryption & Authentication, Firewall protection, and Security Best Practices.

These checks help protect the computer from unauthorized access, malware, and other cyber threats.

2. Encryption & Authentication

Encryption and authentication ensure that only authorized users can access the system and sensitive information.

Security Check Performed

Verified that the computer requires a password for login

Checked that a secure password is created

Confirmed that Windows sign-in authentication is active
